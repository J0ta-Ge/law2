Jota

Simple Text Editor for Android

Jota is a simple text editor for Android.

It is designed to be lightweight and fast, with a focus on simplicity and ease of use.

Jota supports syntax highlighting for many programming languages, making it a great choice for coding on the go.

It also includes features like auto-completion, search and replace, and customizable themes.

Jota is available for free on the Google Play Store, with a pro version that includes additional features.

For more information, visit the Jota page on Google Play.
